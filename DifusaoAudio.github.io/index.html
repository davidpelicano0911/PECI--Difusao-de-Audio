Contexto

A gestão eficiente de áudio é essencial em espaços como hotéis, centros comerciais, restaurantes e lojas onde diferentes zonas requerem anúncios personalizados ou uma simples música de ambiente. Este projeto pretende responder a estas necessidades através de um sistema wireless composto por um nó central de controlo e múltiplos recetores distribuidos. Para isso,temos como objetivo aumentar a flexibilidade na gestão de áudio, permitindo uma experiência adaptada a cada zona e garantindo qualidade, simplicidade e eficiência na operação.


Motivação

A gestão de múltiplas zonas com diferentes necessidades sonoras, quando feita através de sistemas tradicionais baseados em cabos, é frequentemente complexa e pouco flexível. O uso de cabos exige infraestrutura física extensa, instalação trabalhosa e custos elevados de manutenção. Este projeto propõe uma solução wireless e flexível, eliminando a necessidade de cabeamento. Assim, é possível assegurar uma distribuição de áudio intuitiva e com maior facilidade para expansões ou reconfigurações.


Objetivos


Desenhar a Unidade de Controlo Central

Adicionar Capacidades Sem Fios

Desenhar o Recetor

Reduzir os efeitos da latência e das interrupções.


Resultados Esperados


Para atingir os objetivos propostos, esperamos alcançar os seguintes resultados:

Sincronização

Áudio sincronizado entre as zonas para uma experiência consistente.

Interface para Controlo Remoto

Interface intuitiva, permitindo fácil acesso a partir de dispositivos móveis ou desktops.

Qualidade de Áudio

O sistema assegura uma boa experiência de áudio, sem interrupções ou latência percetível.
